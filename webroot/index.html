Welcome to Simple HTTP Server

This server is built with pure Java Socket API

Features

  • ✅ Pure Java Socket implementation (no frameworks)
  • ✅ GET and POST method support
  • ✅ HTTP/1.1 with keep-alive connections
  • ✅ Multiple status codes (200, 301, 302, 304, 404, 405, 500)
  • ✅ Multiple MIME types (HTML, JSON, JPEG, etc.)
  • ✅ User registration and login API
  • ✅ Redirect handling in client

API Endpoints

User Registration

POST /api/register

Parameters: username, password

User Login

POST /api/login

Parameters: username, password

Example using curl:

curl -X POST http://localhost:8080/api/register \
  -d "username=newuser&password=mypassword"
